# Indic Manuscript Auto Annotation

AI-powered system to generate pre-annotations for Indic manuscripts to reduce manual labeling effort.

## Problem
Manual annotation of manuscripts is slow and expensive.

## Solution
AI-assisted annotation pipeline that:
- detects layout regions
- extracts text lines
- generates structured annotation files
- allows human correction

## Tech Stack
Frontend: React
Backend: FastAPI
AI: YOLO / Detectron2 / EasyOCR
